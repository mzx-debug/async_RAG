```bash

  --xE 0 --xR 0 --b 32 --sample-queries 256 \
  --pipeline-mode async_v2 \
  --index-path indexes/beir_nfcorpus/faiss.index \
  --corpus-path data/beir_nfcorpus/corpus.jsonl \
  --queries-file data/beir_nfcorpus/queries.jsonl \
  --generator-model Qwen/Qwen2.5-1.5B-Instruct \
  --output-json output/run.json \
  --gpu-memory-utilization 0.80 --gpu-id 0
```

## 核心概念

**(xE, xR)**：embedding/retrieval 设备。0=CPU, 1=GPU。

**成本模型**：
```
# 1. 每批次生成时间（双曲线模型）
gen_q[(xE,xR)] = (P_fixed / B + D_fixed) * contention[(xE,xR)]
  P_fixed  (ms):    固定 prefill/launch 开销
  D_fixed  (ms/q):  每 query decode 边际成本
  contention:        运行时 EMA，obs/pred 比值，吸收模型不匹配和 GPU 争用

# 2. 设备执行时间（总时间，per-batch）
emb_per_token[xE] = emb_per_token_ema[xE]          # ms/token
L̄_in              = 本批次平均输入长度（token）
emb_total          = emb_per_token[xE] * L̄_in * B  # 随序列长度缩放
ret_total[xR]     = retrieval_latency_ema[xR]        # FAISS 批量查一次，不乘 B
er_base           = er_base_overhead_ema             # ms（E+R 启动开销，≈0）

# 3. wall_q = wall_total / B（per-query）
#    其中 gen_total = gen_q * B, emb_total = emb_per_token * L̄_in * B
(0,0): max(gen_total, er_base + emb_total + ret_total) / B   # CPU E+R || GPU gen
(1,0): (er_base + emb_total + max(ret_total, gen_total)) / B
(0,1): (er_base + ret_total + max(emb_total, gen_total)) / B
(1,1): (er_base + emb_total + ret_total + gen_total) / B

所有参数通过 EMA 从实际执行数据中持续校准。
```

## 已知问题 & 解决

### CUDA 不可用
- **现象**：`torch.cuda.is_available()` 返回 False
- **原因**：未安装 CUDA 版 PyTorch，或 conda 环境未激活
- **解决**：`pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124`

### vLLM 未安装
- **现象**：`No module named 'vllm'`
- **解决**：`pip install vllm>=0.6.0`

### FAISS GPU OOM（cublas error 13）
- **现象**：`Assertion 'err == CUBLAS_STATUS_SUCCESS' failed in runMatrixMult`
- **原因**：vLLM 显存占用过高，FAISS GPU 矩阵运算无足够 VRAM
- **解决**：降低 `--gpu-memory-utilization` 至 0.45，或用更小的生成模型

### FlashInfer JIT 失败
- **现象**：`RuntimeError: Ninja build failed...`
- **解决**：`rm -rf ~/.cache/flashinfer/`

### 模型下载慢/失败
- **解决**：`export HF_ENDPOINT=https://hf-mirror.com`
